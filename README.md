
## Installation

Install the following packages

**Official Repo (w/ Pacman)**
- hyprland
- waybar
- pamixer
- playerctl
- pavucontrol
- brightnessctl
- ttf-font-awesome
- ttf-jetbrains-mono
- ttf-iosevka-nerd
- ttf-opensans
- nwg-look
- qt5ct
- qt6ct
- ttf-hack
- kvantum
- breeze-icons

**AUR**
- wlogout
- numix-gtk-theme-git 


Copy the folder hypr under ~/.config/hypr
Copy the folder waybar under ~/.config/waybar
