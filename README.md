
## Preparation

1. Install ArchLinux
2. install the following packages

**Official Repo (w/ Pacman)**
- hyprland
- waybar
- pamixer
- playerctl
- pavucontrol
- brightnessctl
- ttf-font-awesome
- ttf-jetbrains-mono
- ttf-iosevka-nerd
- ttf-opensans
- nwg-look
- qt5ct
- qt6ct
- ttf-hack
- kvantum
- breeze-icons

**AUR**
- wlogout
- numix-gtk-theme-git 


## Dot Files Installation

1. Configure **hyprland** by copying the folder hypr under ~/.config/hypr
2. Configure **waybar** by copying the folder waybar under ~/.config/waybar
